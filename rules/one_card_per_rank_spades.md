# EN 

Cards for this suit were dealt in such a way that each player hold exactly one ace, exactly one king, ... , exactly one 2.
The dealer is playing a ♠ contract without any bidding and scores +100 for every trick they make.

If you're using bridgemate input passed out if declarer made 0 tricks, or 7♠ by opponents down appropriate number of tricks to match the score.


# PL

Karty do tego rozdania zostały potasowane w taki sposób, żeby każdy z graczy miał dokładnie jednego asa, dokładnie jednego króla, ..., dokładnie jedną dwójkę.
Dealer rozgrywa kontrakt w ♠ bez licytacji i dostaje +100 za każdą wziętą lewę.

Jeśli wpisujecie zapis do pierniczka, wpiszcie 4 pasy jeśli wziął 0, lub 7♠ przez przeciwników bez odpowiedniej liczby lew, żeby wynik się zgadzał.
